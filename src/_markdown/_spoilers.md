## Spoilers

Hide spoilers by adding an angle bracket and an exclamation point `>!` before each line of text. 

```
>! This is a spoiler.

>! Still spoiling!

Not a spoiler.

>! A different spoiler.

>! >! You can even nest a spoiler within a spoiler!
```
