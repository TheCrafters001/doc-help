## Vimeo Video

Embed a video from [Vimeo](https://vimeo.com).

#### Syntax

{% raw %}
```
{% vimeo videoID %}
```
{% endraw %}

#### Required Parameters

- **videoID**
  You can find the video ID at the end of video's URL. For example, the Vimeo video ID of `http://vimeo.com/2619976` is `2619976`.

#### Example

{% raw %}
```
{% vimeo 2619976 %}
```
{% endraw %}
