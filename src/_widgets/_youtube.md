## YouTube Video

Embed a video from [YouTube](https://youtube.com).

#### Syntax

{% raw %}
```
{% youtube videoID %}
```
{% endraw %}

#### Required Parameters

- **videoID**
  You can find the video ID at the end of the video's URL. For example, the YouTube video ID of `https://www.youtube.com/watch?v=dQw4w9WgXcQ` is `dQw4w9WgXcQ`.

#### Example

{% raw %}
```
{% youtube dQw4w9WgXcQ %}
```
{% endraw %}
